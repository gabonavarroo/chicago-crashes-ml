# ML Workstream

This directory contains planning, SQL pipelines, model training code, configuration, and artifacts for crash severity and hotspot prediction.

## Structure

- `plans/`: planning docs and operating protocols
- `sql/`: feature and prediction table build SQL
- `src/`: extraction, feature engineering, training, evaluation, and scoring code
- `configs/`: model and policy configuration
- `artifacts/`: trained model binaries, calibration artifacts, and metadata
